# test task

Тестовий таск з компонентом відгуків

# Технології:

Vue, Vite, TypeScript, NuxtUI, Axios.

# Запуск проєкту

1. Встановити залежності: npm i
2. Запустити проєкт: npm run dev
3. Відкрити в браузері http://localhost:5173/

# Про api

Тут використовується mockapi.io, створено ендпоінт reviews, який повертає список айтемів всіх оглядів
